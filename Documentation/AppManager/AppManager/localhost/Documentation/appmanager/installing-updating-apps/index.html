Installing & Updating Apps

The installation button is located at the lower left side of the screen, at the bottom of the app list.

Clicking the “Install” button will open a file browser dialog. Select an valid application package file (.apf), created by APFCreator. The file will be transfered to the target device for installation.

You will also be asked if you want to force the installation. Applications normally cannot be installed if there is already an application with the same application identifer and the same or a higher version as the application to be installed. The force flag will dispass this restriction.
